# coc-list-vim-select

vim.ui.select() support backed by coc-list

## Install

`:CocInstall coc-list-vim-select`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-list-vim-select-keymap)`

## Lists

`:CocList demo_list`
